# Documentação do Azure Functions

O Azure Functions é um serviço de nuvem disponível sob demanda que fornece toda a infraestrutura e os recursos continuamente atualizados necessários para executar os aplicativos. Você se concentra no código que mais importa para você, na linguagem mais produtiva para você, e o Functions manipula o restante. O Functions fornece computação sem servidor ao Azure. Você pode usar o Functions para criar APIs Web, responder a alterações no banco de dados, processar fluxos de IoT, gerenciar filas de mensagens, entre outras ações.
